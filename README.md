# PROJET YDAYS : FACTEURS DE RISQUE POUR ASTEROÏDES

## Objectifs

Réaliser une analyse exploratoire et un document interactif (type PowerBI) pour évaluer les facteurs qui rendent un objet astronomique (geocroiseur) dangereux pour la Terre.
On finira le projet par une conclusion en se servant à la fois des données exploitées dans une API fournie par la NASA et des connaissances personnelles pour établir les facteurs de risque.
